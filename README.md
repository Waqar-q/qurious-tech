# Qurious Tech — Landing Page

Powering businesses with custom websites, smart automation, and creative branding.

Built with plain HTML/CSS/JS — ready to deploy on GitHub Pages.

## Deploy to GitHub Pages

1. Create a repo on GitHub (e.g., `qurious-tech-landing`)
2. Push these files to the `main` branch
3. Go to **Settings → Pages** → select `main` branch → Save
4. Your site will be live at `https://waqar-q.github.io/qurious-tech-landing`

## Make the Form Work

1. Go to [Web3Forms](https://web3forms.com) and sign up for free
2. Create a new form to get your **Access Key**
3. Open `index.html` and replace `YOUR_WEB3FORMS_ACCESS_KEY` with your actual key

```html
<input type="hidden" name="access_key" value="your-actual-key-here" />
```

That's it — form submissions will be emailed to you.
